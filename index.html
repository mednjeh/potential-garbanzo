br

تعمل سطر جديد (line break). تُكتب هكا: <br>

p

تكتب بيها فقرة نصية. تُكتب هكا: <p>...</p>

h1 → h6

للعناوين. <h1> أكبر و<h6> أصغر.

strong

تُغلّظ النص وتعني أنه مهم (semantic).

mark

تعمل highlight على النص.

em

تميل النص (italic) وتعني تأكيد.

ul

قائمة غير مرتبة (●). تُكتب: <ul><li>...</li></ul>

ol

قائمة مرتبة (1,2,3...). تنجم تغيّر النوع بـ type.

li

عنصر داخل ul أو ol.

dl

قائمة تعريف فيها dt و dd.

dt

المصطلح أو الشيء اللي باش تفسروه.

dd

التفسير للمصطلح داخل dl.

img

لعرض صورة. مثال: <img src="..." alt="...">

a

رابط داخلي أو خارجي. يمكن تربطه بصورة أو نص.

table

لإنشاء جدول. يُستخدم مع tr، td، th.

tr

صف من الجدول. يحتوي خلايا td أو th.

th

رأس عمود أو صف (bold & centered).

td

خلية بيانات داخل الجدول.

label

نص يفسّر input ويرتبط بيه عن طريق for.

input

خانة إدخال (نص، كلمة سر، checkbox، إلخ).

fieldset

مجموعة عناصر داخل فورم. يمكن تضيف legend.

css

تُستعمل لتزيين الصفحة: الألوان، المسافات، الخطوط... يمكن إضافتها داخلياً، خارجياً أو مباشرة.
